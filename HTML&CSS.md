### HTML

#### 注释

1. 格式`<!-- 注释内容 -->`
2. vscode 中的快捷键`ctrl + /`

#### 元素

1. 元素 = 起始标记 + 元素属性 + 元素内容 + 结束标记
   - `<a href="http://www.baidu.com">百度一下</a>`
   - `<div class="wrapper"></div>`
2. 有些元素是没有结束标记的
   - `<meta charset="UTF-8" />`
   - `<img src="" alt="">`
3. 不同的元素有不同的功能
